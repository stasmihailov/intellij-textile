h2. Textile language support

"!https://img.shields.io/jetbrains/plugin/d/com.potniype4kin.intellij-textile(JetBrains IntelliJ plugins)!":https://plugins.jetbrains.com/plugin/14204-textile-language-support

Adds support for Textile markup to Jetbrains IDEs

Main features:
- adds Confluence-like code highlighing
- adds ability to generate HTML from textile document by "Generate HTML from this file" intention action
